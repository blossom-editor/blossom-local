Blossom-local 是一个基于 Blossom 的本地版本, 所有文件均保存在本地，不做任何破坏性的改动，文件关系实时映射。

支持 Windows。

<p align="center">
<img src="./doc/imgs/device.png">
</p>

<br/><br/>

# 👏 Blossom 的特点:

### 完善的文件关系

基于 Markdown 编写，没有破坏性的语法拓展，在这里编写的内容在任何 Markdown 软件中都能正常显示。

### 丰富的附加功能

- 番茄钟
- 图片管理
- 字数统计、编辑热力图、天气预报、主题设置...
